# pie-safe Gleam Monorepo ŌĆö Hello World Implementation Plan

## Phase 0 ŌĆö Repo housekeeping
- Create a root `.gitignore` covering `*/build/`, `*/.lustre/`, `*/manifest.toml`, `backend/data/`

## Phase 1 ŌĆö `core` package
- `gleam new core`
- Delete generated `src/core.gleam`; create `src/core/greeting.gleam` with `pub type Greeting { Greeting(message: String) }`
- No target lock; only `gleam_stdlib` dependency
- `gleam build` to verify

## Phase 2 ŌĆö `ui` package (JavaScript target)
- `gleam new ui`
- Replace `gleam.toml`: `target = "javascript"`, add `lustre`, `lustre_dev_tools` (dev), `core = { path = "../core" }`, add `[tools.lustre.build] outdir = "../backend/priv/static"`
- Write `src/ui.gleam`: `lustre.simple` SPA that creates `Greeting(message: "Hello, World!")` and renders it in a centered Tailwind-styled `<h1>`
- Tailwind auto-detected by `lustre_dev_tools` from the utility classes in source
- `gleam build` to verify

## Phase 3 ŌĆö Prep backend static dir
- `mkdir -p backend/priv/static`

## Phase 4 ŌĆö `backend` package (Erlang target)
- `gleam new backend`
- Replace `gleam.toml`: `target = "erlang"`, add `mist`, `gleam_erlang`, `gleam_http`, `sqlight`, `parrot`, `core = { path = "../core" }`
- Create `src/sql/schema.sql` (greetings table) and `src/sql/queries.sql` (ListGreetings :many)
- `mkdir -p backend/data && sqlite3 backend/data/app.db < backend/src/sql/schema.sql` to seed the DB for Parrot introspection
- Run `gleam run -m parrot -- --sqlite data/app.db` inside `backend/` ŌåÆ generates `src/backend/sql.gleam` (committed)
- Write `src/backend.gleam`: Mist server on port 3000 ŌĆö `GET /` returns inline `index.html` loading `/static/ui.js` + `/static/ui.css`; `GET /static/*` serves files from `priv/static/` via `mist.send_file`; startup calls `ListGreetings` and logs the count
- SQLite connection is held open via `sqlight.with_connection` for the process lifetime
- `gleam build` to verify

## Phase 5 ŌĆö End-to-end verification
- Inside `ui/`: `gleam run -m lustre/dev build --outdir=../backend/priv/static` ŌåÆ produces `ui.js` + `ui.css`
- Inside `backend/`: `gleam run` ŌåÆ server starts on port 3000
- `http://localhost:3000` shows Tailwind-styled **"Hello, World!"** heading
- Terminal logs: `Startup DB query: found 0 greeting(s)`

## Phase 6 ŌĆö Tidy-up
- Confirm `backend/data/app.db` is gitignored
- Confirm `backend/src/backend/sql.gleam` is tracked in git
